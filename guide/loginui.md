# LoginUI

El componente `LoginUI` es un componente de interfaz de usuario de JettraWUI que genera la etiqueta `<loginui>`. Se utiliza para encapsular las interfaces de autenticación, integrándose de forma nativa con los diseños y el framework.

## Uso Básico

```java
import io.jettra.wui.components.LoginUI;

LoginUI loginui = new LoginUI();
loginui.setProperty("id", "myLoginUI");
```

## Ejemplo

Para ver un ejemplo funcional y el código generado, consulta la página de demostración `LoginUIPage` en el proyecto `JettraWebExample`.
