---
title: Calendar
description: Componente para mostrar calendarios mensuales
package: io.jettra.wui.components.Calendar
---

El componente `Calendar` provee una vista clásica en modo grilla para un mes. Ideal para marcar días, ver fechas globalmente o planificar.

### Uso Básico

```java
import io.jettra.wui.components.Calendar;

Calendar cal = new Calendar();
cal.setStyle("margin-top", "20px");
```
